About Block Site

Block Site enables you to stay focused and improve your productivity. You can easily block any distracting or harmful website. No willpower necessary.

Create Block Lists

Add the websites or pages you want to block to your Site List and get automatically redirected to safer, more positive sites. You can also create a list of friendly sites, and switch from blocking specific sites to only allowing certain sites to work.

Right Click Menu

Right click anywhere on any website to open the quick access menu. Block sites in seconds to avoid procrastination and stay on target.

Incognito mode

All the Block Site functionality can also work in the incognito mode, you just need to allow it. Follow these instructions to see how it is done: incognito mode

Keep Your Kids’ Internet Safe

Activate “Adult related” keyword block to filter out all predefined bad words and profanities. Highly recommended for parents with young children.

Block For Specific Times

Choose days and intervals in which you want Block site to be active. Set up custom blocks based on your work schedule (for example: 09:00 - 17:00), so you won’t procrastinate during the day, when at work or at school.

Set Up Password

Set Up a password in the Settings page to limit the access to Block Site Options, or to enable the option to access blocked sites with a password.

How to Run an Extension in Incognito Mode

Quick and simple manual how to run any of our extensions in Incognito Mode

Simple:

  1. Go to the list of your extensions: chrome://extensions/
  2. Locate the extension you want to allow in incognito.
  3. Check the "Allow in Incognito"

Note: You will get the warning message, but this is just informative. Enjoy.
